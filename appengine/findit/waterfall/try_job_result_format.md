The format for recipe result for compile failures is:
    {
        'result': {
            'rev1': 'passed',
            'rev2': 'failed'
        },
        'metadata': {}
    }


The format for final try-job result for compile failures saved in WfTryJob.compile_results is:
    [
        {
            'report': {
                'result': {
                    'rev1': 'passed',
                    'rev2': 'failed'
                },
                'metadata': {}
            },
            'url': 'url',
            'try_job_id': '1',
            'culprit': {
                'revision': 'rev2',
                'commit_position': '2',
                'review_url': 'url_2'
            }
        },
        ...
    ]
